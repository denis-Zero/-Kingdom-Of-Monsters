# Kingdom Of Monsters

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse:
```
http://localhost:3000
```

## Deploy grátis (Render)

1. Suba o projeto no GitHub.
2. Entre em https://render.com e crie um serviço Web.
3. Selecione o repositório.
4. Render detecta `render.yaml` e usa:
   - Build: `npm install`
   - Start: `npm start`
5. Depois de publicar, use a URL do Render para jogar online.

Observação: o plano grátis pode “dormir” após inatividade.
